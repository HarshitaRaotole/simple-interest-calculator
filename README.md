# Simple Interest Calculator

## Description
A simple command-line tool or script that calculates the simple interest earned or paid over a specific period based on a principal amount and an annual interest rate.

## Calculation Formula
The formula used for calculating simple interest is:
**I = P * R * T / 100**

Where:
*   **I** = Interest
*   **P** = Principal Amount (Initial investment or loan amount)
*   **R** = Annual Interest Rate (as a percentage)
*   **T** = Time period (in years)

The total amount (Principal + Interest) is calculated as:
**Total = P + I**

## Features
- Input fields for Principal, Rate, and Time.
- Instant calculation of total interest.
- Displays the final total amount.

## How to Use
1. Clone this repository to your local machine.
2. Run the script (e.g., `python simple_interest.py`).
3. Enter the required values when prompted.
4. The calculated interest and total amount will be displayed on the console.
